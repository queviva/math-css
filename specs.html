    x2 3 5 + 9
    allows for extremely simple fraktions, or more complicated if needed

    data-cank // draws cancel slash over it
    data-opt# // alternative shapes
    data-flip // flips the circle
    data-turn // rotates the circle

  • **exp shorthand for exponent
    x**2 y**(3 + Xi) z**x**2
    allows for exponents using ** intead of <sup></sup> tags

    requires mathrobatics.js and must be enclosed in a <b> tag

    if you want to actually type ** after a werd character, escape it with a backslash; use ()'s, []'s, or an html tag to include spaces in the exponent 
    <b> <!-- must be in a b-tab -->
    x\**2
    x\**(3 + 5)
    x\**[7 - y]
    x\**<v-iva>7<v-iva>
    x\\** <!-- literal display of ** -->
</b>
